### 背景

有时我们在使用 git 的时候，会在本地生成文件夹然后关联到 github 上面的远程仓库。所以这里简单说下 git 的命令和关联步骤。<!--more-->

1. 本地生成文件夹

```
mkdir test
```

2. 进入和 git 初始化本地仓库文件夹

```
cd test
git init
```

3. git 添加所有文夹

```
git add .
```

4. git 提交所有文件

```
git commit -m "这里是log日志" -a
```

5. github 上建立一个远程仓库

6. 复制远程仓库的地址

7. 连接到远程仓库

```
git remote add origin https://github.com/wangyuan0108/***.git
```

8. 把本地项目推送到远程仓库

```
git push -u origin master
```
